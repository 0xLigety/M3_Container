FORTE on a MRX

This is a demonstration container running the 4DIAC runtime environment called FORTE. This container can be programmed with the corresponding IDE to set up a Soft-SPS on the MRX device. To afford you the use of the MRX functionality the following functionblocks were created:
  • a general cli block to send single commands to the command line interface
  • an output block to control the outputs (if you are using at least one MRXcard SI)
  • an input block to get the status of digital inputs
  • an email block to send messages via email
  • a sms block to send messages via sms
  • a restart block to reboot the device if problems occured
  • an info-led block to use the info-LED of your MRX device

In order to program the SPS please follow these steps:
  1. Download the corresponding IDE for you operating system
  2. Click here to download the INSYS-Functionblocks
  3. Unzip the folder
  4. Move folder to "Path to your 4DIAC-IDE"/typelibrary
