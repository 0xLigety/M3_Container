This container demonstrates the MRX as a MQTT-Broker. MQTT is a new communication method to send data from clients to a broker which remits them to other clients.


What you need

1. MRX device

2. Network connection

3. MQTT Client (for example MQTTLens)

Configuration

Configure the gateway and dns (/Container/IP-Network)

Change user and password for the WEB-Interface (/Container/Athentication)

Set up the mqtt broker (/Application/Configuration)
