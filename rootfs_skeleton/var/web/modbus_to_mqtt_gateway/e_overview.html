This container demonstrates the connection of a modbus device and sending the data via mqtt. MQTT is a new communication method to send data via a mrx device to cloud-services or your own broker.


What you need

1. Modbus device

2. MQTT-Broker

3. MRX

Configuration

Configure the gateway and dns (/Container/IP-Network)

Change user and password for the WEB-Interface (/Container/Athentication)

Set up the modbus device und your mqtt broker (/Application/Configuration)
