This container contains tools for analyzing and debugging networking such as
  • iputils (with full blown ping, rarpd, rdisc, tracepath, traceroute, traceroute6)
  • busybox tools (like nc, netstat, ip, servers and many more
  • httping
  • iperf
  • curl
  • mcip
  • mcip-tool
  • openvpn
  • strace
  • stunnel
  • tcpdump
  • iptables
It runs an SSH server you can access on its IP address on port 22.

User name: root
Password: root
